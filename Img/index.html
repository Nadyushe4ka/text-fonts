Object-fit

Css-свойство object-fit определяет, как содержимое изменяемого элемента должно заполнять контейнер относительно его высоты и ширины.

Документация на MDN

Замена этого свойства на svg для поддержки устаревших браузеров

Поддержка браузерами
fill
contain
cover
scale-down
none

img {
  width: 100%;
  height: auto;
  height: 256px;
}
